# Experiments

This repository contains a **paper-focused experiment harness** under:

- `experiments/capo_paper/`

It provides a simple workflow:

1) run a small diagnostic on an A10 (fast sanity check),
2) run the paper-minimum suite (E1–E4),
3) build all paper figures/tables from run logs.

Start here:

- `experiments/capo_paper/README.md`
